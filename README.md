Цель проекта - разработать команды и аргументы для управления БД с возможностью автоматического запроса цен на определенный тип предметов игры Counter-Strike 2

Проект можно модицифировать в зависимости от нужд пользователя, для этого нужно модифицировать класс Parser с его функциями для подгона под типы и имена элементов.
Проект утилизирует систему с несколькими файлами для разделенеления синтакса библиотек и структуризации кода
Для работы проекта необходим драйвер браузера Microsoft Edge, у проекта есть requirements.txt
